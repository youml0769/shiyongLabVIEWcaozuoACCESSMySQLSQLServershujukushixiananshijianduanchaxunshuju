## 使用说明

1. **环境准备**：确保您的计算机上已安装LabVIEW开发环境，并且已配置好相应的数据库驱动程序。
2. **代码导入**：将资源文件中的LabVIEW代码导入到您的项目中。
3. **配置数据库连接**：根据您的数据库类型和连接信息，修改代码中的数据库连接参数。
